Implemented the following with OpenCode:
- Mood Calendar
- Monthly/Yearly Trend Charts
- Login Screen
- Data Encryption
- Dark Mode
- Multi-language support
- Water Intake Tracker

Implemented the following with Copilot:
- Sleep Tracker

Process was:
1. Ask AI implement this feature
2. Review code to gauge general usefulness
3. Manual testing to evaluate shortcomings
4. Repeat until satisfied
5. Ask Another AI to gauge how well the implementation was done for certain parts that I was uncertain about. 

Initially used ChatGPT and Gemini for some guideance on how to implement certain features, but the output was too generic.
I moved on to use Copilot, but encountered issues with its implementation due to context.

OpenCode would implement things completely and seemed to be capable of iteratively improving as I found problems with its implementation.
